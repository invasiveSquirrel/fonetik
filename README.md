# Fonetik: Master the Building Blocks of Language

Fonetik is a specialized tool for mastering the most difficult sounds in a new language. It uses the International Phonetic Alphabet (IPA) and high-fidelity audio to show you exactly how to pronounce tricky phonemes.

## Key Features
- **Phonetic Focus:** Study individual sounds rather than just words.
- **Complex Sound Sets:** Includes specialized data for languages like Scottish Gaelic, covering broad/slender distinctions and pre-aspiration.
- **Premium Audio:** Uses the same high-quality neural voices as Wordhord to demonstrate perfect pronunciation.
- **Multilingual:** Supports a wide range of English dialects and European languages.

## Installation Instructions

### 🐧 Linux
1. Open your terminal in this folder.
2. Run `./fonetik-launcher.sh`.

### 🪟 Windows
1. Install [Node.js](https://nodejs.org).
2. Click **"Code" > "Download ZIP"** on GitHub and unzip it.
3. In the folder, hold **Shift + Right-click** and select **"Open PowerShell window here."**
4. Type `npm install` and press Enter.
5. Type `npm start` to run.

### 🍎 Mac
1. Install [Node.js](https://nodejs.org).
2. Download and unzip the code.
3. Open **Terminal**, drag the folder into the window, and press Enter.
4. Type `npm install` and press Enter.
5. Type `npm start` to run.
